# Power BI Dashboards

This folder contains Power BI reports and dashboards for business intelligence and data visualization projects.  

## Skills Demonstrated
- Data modeling and relationship building
- Measures and calculations using DAX
- Interactive reports with filters, slicers, and visuals
- Data storytelling and actionable insights


Each dashboard includes:
- `.pbix` files
- Explanation of metrics and visualizations
- Screenshots or PDF exports of the report for reference
